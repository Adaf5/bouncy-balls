# Bouncy Balls!

A simple SDL3 bouncing balls exercise.

## Dependencies
- MinGW-w64 (g++)
- SDL3
- SDL_mixer 3.0

## Building
1. Install MinGW and SDL3 as described in setup
2. Run windres to compile the icon resource:
   windres icon.rc -o icon.o
3. come one you know how to build stuff if you're here

## Controls
- Left click: spawn ball
- Up/Down arrows: change speed
- F: toggle fullscreen (Kinda clanky)
- Escape: kills your grandma
